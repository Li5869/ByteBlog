# 博主探索 Skill

## 概述

本 Skill 提供博客平台的**博主搜索和发现**能力。涵盖关键词搜索博主、热门博主排行、博主详细信息查看。适用于用户寻找特定领域博主、了解博主影响力或关注作者动态的场景。

## 触发条件

当用户输入涉及以下意图时，调用本 Skill：

- 搜索特定博主（如"帮我找找张三这个博主"）
- 查看热门/优质博主（如"有哪些受欢迎的博主"）
- 获取博主详细信息（如"看看这个博主的资料"）
- 寻找特定领域的内容创作者（如"有没有写 Java 的博主"）

## 使用到的工具

| 工具函数 | 用途 | 来源文件 |
|---------|------|---------|
| `search_authors_by_keyword(keyword, limit=5)` | 根据关键词搜索博主（昵称 boost 2.0 + 简介 + 用户名） | `tools/author_tool.py` |
| `get_hot_authors(limit=10)` | 获取粉丝数最多的热门博主（仅展示有文章的活跃博主） | `tools/author_tool.py` |
| `get_author_by_id(author_id)` | 获取博主详细信息（文章数、粉丝数、获赞数、个人简介等） | `tools/author_tool.py` |
| `search_articles_by_keyword(keyword, limit=5)` | 结合搜索：查找该博主的热门文章 | `tools/article_tool.py` |

## 工作流程

### 场景一：搜索博主

```
用户请求 → search_authors_by_keyword → 格式化结果 → 返回博主列表
```

1. 从用户请求中提取博主名称或关键词
2. 调用 `search_authors_by_keyword(keyword, limit=5)` 进行搜索
3. 返回匹配的博主列表（昵称、简介、粉丝数、文章数）

### 场景二：热门博主推荐

```
用户请求 → get_hot_authors → 格式化结果 → 返回热门博主排行
```

1. 调用 `get_hot_authors(limit=10)` 获取热门博主
2. 按粉丝数降序展示
3. 可附带每位博主的核心信息（文章数、获赞数）

### 场景三：查看博主详情

```
用户请求 → get_author_by_id → (可选) 搜索该博主文章 → 返回详情
```

1. 从用户请求中提取博主 ID
2. 调用 `get_author_by_id(author_id)` 获取详细信息
3. 若用户想了解博主的文章，可调用 `search_articles_by_keyword(nickname)` 查找其作品

## 使用示例

### 搜索博主

```
用户：帮我搜索一下写 AI 相关的博主
AI：正在搜索相关博主...
[调用 search_authors_by_keyword("AI", limit=5)]
为您找到以下博主：
1. 👤 张三 - 📝 45 篇文章 - ❤️ 1,234 粉丝
   📋 简介：专注 AI 和机器学习领域
2. 👤 李四 - 📝 28 篇文章 - ❤️ 856 粉丝
   📋 简介：LLM 应用开发实践者
```

### 热门博主排行

```
用户：有哪些受欢迎的博主？
AI：正在获取热门博主排行...
[调用 get_hot_authors(limit=10)]
以下是热门博主 TOP 10：

...
```

### 查看博主详情

用户：ID 是 1001

[调用 get_author_by_id(1001)]
👤 博主：张三

## 注意事项

- 搜索结果按相关性评分降序排列，匹配昵称权重最高
- 热门博主仅展示有文章的活跃博主（文章数 > 0）
- 查看博主详情后，可主动询问用户是否需要进一步查看该博主的文章
- 当搜索结果为空时，建议换关键词或查看热门博主
